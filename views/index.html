Buzz Circle API

Routes:

Posts
  • GET/posts  ➜   Get all posts
  • GET/posts/?page=<pageNumber> ➜   Get a page of posts
  • GET/posts/?userId=<userId>  ➜   Get posts with userId <userId>
  • POST/posts ➜   Add new post
  • GET/posts/:postId ➜   Get post with id <postId>
  • PUT/posts/:postId ➜   Update post
  • DELETE/posts/:postId ➜   Delete post
  • PATCH/posts/:postId ➜   Update post reactions
Users
  • GET/users ➜   Get all users
  • GET/users/?page=<pageNumber>➜   Get a page of users
  • GET/users/?username=<username> ➜   Get user with username <username>
  • GET/users/:userId ➜   Get user with id <userId>
  • DELETE/users/:userId ➜   Delete user
